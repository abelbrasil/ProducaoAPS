# 📊 Painel de Produção APS

Dashboard interativo desenvolvido com Streamlit para análise da produção de exames e procedimentos na Atenção Primária à Saúde (APS).

## 🚀 Tecnologias

- Python
- Streamlit
- Supabase (PostgreSQL)
- Pandas
- Plotly

## 📈 Indicadores

- Exames Solicitados
- Exames Avaliados
- Quantidade de Exames
- Procedimentos
- Média mensal dos indicadores
- % de Exames Avaliados

## 🔗 Acesso

https://relatorioaps.streamlit.app/

## ⚙️ Arquitetura

- Pipeline ETL em Python
- Armazenamento em Supabase
- Camada de visualização em Streamlit
